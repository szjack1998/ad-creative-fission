# 新手使用指南（小白专用）

> 目标：让你在一台**全新电脑**上，从零把「爆款素材拆解裂变器」跑起来，用你**自己腾讯广告账户里的真实素材**产出裂变报告。
> 全程不需要懂代码、不需要手动导出 Excel/CSV。你只负责「装好依赖 + 给我数据权限」，剩下的脚本全自动。

---

## 一、先搞清楚：这套东西在干嘛

你投了腾讯广告（朋友圈 / 视频号 / 公众号等），账户里有一堆视频素材和消耗数据。
这个 skill 帮你：

1. **自动拉**你账户里的素材 + 消耗（CTR / CVR / 花费 / 转化）
2. **自动拆**每条视频的口播文案（多模态 AI 读图）
3. **自动分析**哪些创意变量有效 / 无效（成因分析）
4. **自动裂变**一批新脚本（钩子 / 人群 / 利益 / 大字报四维度），直接交给剪辑团队

最终产出：两份 HTML 报告（成因分析 + 制片级裂变），打开就能看、就能用。

---

## 二、新电脑最小依赖清单（照着勾 ✅）

| # | 要准备的 | 为什么 | 卡住表现 |
|---|---|---|---|
| ✅ 1 | 安装 WorkBuddy（下载 WB 客户端） | 运行环境 + 这个 skill 的载体 | — |
| ✅ 2 | 安装**两个** skill | ①本 skill `爆款素材拆解裂变器` ②`tencent-ads-assistant`（妙问）。**② 是隐藏依赖**：本 skill 自己不会拉数据，第①步拉素材全靠妙问这个 skill。只装一个跑不起来。 | 拉数据时报"找不到妙问 / 无数据源" |
| ✅ 3 | 一个**已在投**的腾讯广告账户 | 妙问拉的是你账户里**已有的投放数据**。没开户、或没跑过带视频素材+消耗的广告 = 没东西可裂变。 | 拉出来 0 条素材 |
| ✅ 4 | 妙问 API KEY（`sk-mw-xxxxx`） | 去 [miaowen.qq.com](https://miaowen.qq.com/) → 左侧【Skill 社区】→ 复制「你的 API KEY」→ **粘给 WorkBuddy 助手**，助手跑 `setup_token.js` 存到本机。不是填设置框。 | 脚本退出码 2/3（Token 不存在/为空） |
| ✅ 5 | （可选）OpenCV | 纯裂变报告只用 Python 标准库，**不装也能出报告**。只有做「动态视频深度分析」模块时才需 `pip install opencv-python-headless numpy`。 | 跑动态分析时才缺，不影响裂变主流程 |

## 三、操作流程（6 步，从你的视角）

```
[0] 连妙问（你授权） ──▶ [1] 拉素材+消耗 ──▶ [2] 下视频 ──▶ [3] 拆文案 ──▶ [4] 动态分析 ──▶ [5] 出报告
       手动                  全自动            全自动          全自动           全自动            全自动
```

**第 0 步（唯一需要你做的）：连妙问**
- 打开 [miaowen.qq.com](https://miaowen.qq.com/) 登录 → 左侧【Skill 社区】→ 看到「你的 API KEY」（`sk-mw-xxxxx`）
- 点「复制」，把整串粘给 WorkBuddy 助手
- 助手会执行 `node scripts/setup_token.js "<你的KEY>"` 存到本机（无需重启，换电脑要重来一次）

**第 1~5 步：全部让助手跑，你只看结果**
- 助手用妙问 `DATA_QUERY` 自然语言拉「视频素材列表 + 消耗/CTR/CVR」→ 生成 `material_video.json`
- `python batch_extract.py` 下载视频 → `frames_batch/<ID>/video.mp4`
- 多模态自动拆文案 → `videos_multimodal_fixed.csv`
- `python dynamic_analyzer.py`（OpenCV）做动态分析 → `dynamic_analysis.json`
- `python real_fission_v2.py` / `render_dynamic_report.py` → 两份 HTML 报告

> 你只需要对助手说一句："用妙问拉我账户的素材，然后跑完整裂变流程。"
> 助手会自动串起 1~5 步，最后把 HTML 报告给你。

## 四、常见问题（小白自救）

**Q1：拉数据报错 "Token 不存在 / 为空"（退出码 2/3）**
→ 妙问 KEY 没存。回第二步重新粘 KEY 给助手，助手跑 `setup_token.js` 存好。

**Q2：拉出来 0 条素材 / 素材很少**
→ 你账户里可能还没跑过视频广告，或时间范围不对。让助手把时间范围放宽（如"最近 30 天"），确认账户有在投视频素材+消耗。

**Q3：只装了本 skill，没装 tencent-ads-assistant（妙问）**
→ 拉数据那步直接失败。两个 skill 都要装。

**Q4：报告打不开 / 空白**
→ 报告是纯静态 HTML（零 JS），用浏览器直接打开即可。别用奇怪的预览器。

**Q5：想看最终交付长什么样，又不想先连账户？**
→ 让助手用仓库自带的脱敏示例数据跑一遍（`--base ./examples`），先看懂格式再上真数据。

## 五、一句话总结

> **下载 WB → 装 `爆款素材拆解裂变器` + `tencent-ads-assistant` 两个 skill → 账户里先有在投视频广告 → 把妙问 API KEY 粘给助手存好 → 让助手跑「拉数据→裂变」全流程。**

就这五件事，其余全自动。
